Detailed Description

Item to filter different message types.

A filter for message types of the logger. Every message type is assigned one of the following rules:
ShowAll, HideAll, Process
The following message types are distinguished:
trace, debug, info, warning, error, critical, default
Moreover a FilterItem contains a keywords list. If the rule is set to Process the messages of the respective type were scanned for these keywords. Such messages are only shown if they contain at least one of these keywords.

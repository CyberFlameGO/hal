Detailed Description

A gate type contains information about its internals such as input and output pins as well as its Boolean functions.

Member Enumeration Documentation

◆ ClearPresetBehavior

enum hal::GateType::ClearPresetBehavior
strong

Defines the behavior of the gate type in case both clear and preset are active at the same time.

Enumerator
L 

Set the internal state to 0.

H 

Set the internal state to 1.

N 

Do not change the internal state.

T 

Toggle, i.e., invert the internal state.

X 

Set the internal state to X.

undef 

Invalid behavior, used by default.

BooleanFunction hal::netlist_utils::get_subgraph_function ( const std::vector< const Gate * > &  subgraph_gates,
const Net *  output_net 
)

Get the combined Boolean function of a subgraph of the netlist that is defined by a vector of gates and an output net.
The gates within the vector should be connected and the given output net should represent the targeted output of the subgraph.
The resulting Boolean function uses the net IDs of the automatically identified input nets as variables.

Parameters
[in]subgraph_gates- The gates making up the subgraph.
[in]output_net- The output net for which to generate the Boolean function.
Returns
The combined Boolean function of the subgraph.

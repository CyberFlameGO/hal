Gate Type Sequential¶

class hal_py.GateTypeSequential¶

Bases: hal_py.GateType

A sequential gate type contains information about its internals such as input and output pins as well as its Boolean functions. In addition to the standard gate type functionality, it provides mechanisms to specify outputs that depend on the internal state of the sequential gate as well as clock pins.

__init__(self: hal_py.GateTypeSequential, name: str, base_type: hal_py.GateType.BaseType) → None¶

Construct a new LUT gate type by specifying its name and base type. The base type must be either ‘ff’ or ‘latch’.

Parameters:
  • name (str) – The name of the sequential gate type.
  • base_type (hal_py.GateType.BaseType) – The base type of the sequential gate type.

class SetResetBehavior¶

Bases: pybind11_builtins.pybind11_object

Defines the behavior of the gate type in case both set and reset are active at the same time.

Members:

U : Default value when no behavior is specified.

L : Set the internal state to ‘0’.

H : Set the internal state to ‘1’.

N : Do not change the internal state.

T : Toggle, i.e., invert the internal state.

X : Set the internal state to ‘X’.
